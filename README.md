# Chat App
***
Chat App is a realtime MERN Stack chat Application.

## Technologies Used
***
* Nodejs
* Express
* React
* MongoDB
* Socket.io
* Cloudinary API

## Installation
***
A little intro about the installation. 
```
$ git clone https://github.com/Akhil-K-Ponnani/chat-app

**To setup Frontend**
$ cd client
$ npm install
$ npm start

**To setup Backend**
$ cd server
$ npm install
$ npm start
